# Cadastro de Clientes - Projeto A3

Sistema de gerenciamento de clientes desenvolvido como projeto acadêmico utilizando Python e MySQL. O foco principal é realizar operações de CRUD (Create, Read, Update, Delete).

## 🚀 Funcionalidades
* *Cadastro:* Inserir novos clientes com nome, e-mail e telefone.
* *Consulta:* Listar todos os clientes registrados no banco de dados.
* *Atualização:* Editar informações de clientes já existentes.
* *Exclusão:* Remover registros do sistema.

## 🛠️ Tecnologias Utilizadas
* *Linguagem:* Python 3.x
* *Banco de Dados:* MySQL
* *Biblioteca de Conexão:* mysql-connector-python

## 🔧 Como Rodar o Projeto

### Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:
* Python
* MySQL Server
